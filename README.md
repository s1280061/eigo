# IELTS Writing 弱点改善ノート（Kaito 専用）

自分の英作文6本＋公式添削から抽出した文法ミスを **①品詞 ②前置詞 ③冠詞 ④コロケーション ⑤文構造** の5カテゴリ（全48項目）に整理した学習教材。目標は **IELTS Band 6.5**。

## 📱 通勤用ウェブアプリ

**→ https://s1280061.github.io/eigo/**

スマホ最適化。ホーム画面に追加するとアプリのように使えます。

- **📇 カード** … ミスを1枚ずつ復習（📍状況 → ❌→✅、タップで解説）／習得済み・苦手チェック
- **⚡ クイズ** … ❌を見て正しい英語を頭の中で言う → 答え合わせ（進捗・苦手を自動保存、苦手だけ復習も可）
- **📖 模範** … 6設問の模範解答を「① 原文 → ② 自然な表現 → ③ 改良版」で比較
- **🎯 TOP5** … 特に苦手なポイント
- **📅 練習** … 1週間メニュー・優先順位

進捗・苦手はブラウザに保存されます（localStorage、URL・端末ごと）。

## 📄 ファイル

| ファイル | 用途 |
|---|---|
| `data.js` | **単一ソース**（ミス48項目・模範解答6本・TOP5・練習メニュー） |
| `index.html` | 通勤用ウェブアプリ本体（`data.js` を読み込む） |
| `IELTS_弱点改善ノート.md` | ノート全文（`build-note.js` で生成） |
| `IELTS_弱点改善ノート.pdf` | 印刷用（A4・約22ページ、`build-pdf.js` で生成） |
| `build-note.js` / `build-pdf.js` | `data.js` から MD / 印刷HTML を生成 |

## 🔧 再生成（内容を増やしたら）

内容の追加・修正は **`data.js` だけ**編集すれば、アプリ・ノート・PDFすべてに反映されます。

```bash
node build-note.js   # → IELTS_弱点改善ノート.md
node build-pdf.js    # → print.html
# 生成された print.html を Chrome で印刷 → PDF
chrome --headless --print-to-pdf="IELTS_弱点改善ノート.pdf" --no-pdf-header-footer print.html
```
